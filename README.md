# REV Mix AI

> Your music. Mixed by AI.

A high-quality, high-performance music app for Web, Android, and iOS.

## Overview

| | |
|---|---|
| App Name | REV Mix AI |
| Tagline | Your music. Mixed by AI. |
| Platforms | Web, Android, iOS |

## Name Options

1. REV Mix AI (recommended)
2. PulseDJ
3. Tailivo Radio
4. Mixora AI
5. BeatPilot

## Main Goals

- High-quality music streaming
- Fast performance on web and mobile
- AI DJ playlist and mix generation
- Local AI radio
- Offline playback for mobile
- Same account across web, Android, and iOS

## Tech Stack

| Layer | Technology |
|---|---|
| Web | Next.js, React, TypeScript, Tailwind CSS |
| Mobile | React Native with Expo |
| Backend | Node.js with NestJS or Express |
| Database | PostgreSQL |
| Cache | Redis |
| Events | Kafka |
| Storage | S3, MinIO, or Cloudflare R2 |
| Audio | FFmpeg + HLS |
| AI | Ollama, vLLM, or OpenRouter |

## MVP Scope

- Login
- Upload or import music
- Stream songs
- Create playlists
- AI DJ playlist generator
- Web app
- Android app
- iOS app
- Basic radio mode
- Admin panel
